## How to run configurator in your browser?
1. Download [Node.js](https://nodejs.org/en/download/).\
2. Download zip of this repository, unzip the file \
4. Through command prompt navigate to the unzipped file\
5. Type npm install \
6. Type npm run dev \
Have fun!
